What Is HTML

HTML stands for Hyper Text Markup Language. It's a markup language that the web developers use to structure and describe the content of a webpage (not programming language)

HTML consists of elements that describe different types of content: paragraphs , links, headings, images, videos etc web browsers understands. HTML will render HTML codes as websites.

In HTML, each element is ,ade up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
